# Vault Scout Test

pour tester des connexions API de Kiln

## installation

`yarn add dotenv`
ou
`npm install dotenv`

## .env

creer une ficher .env dans le project repertoire avec:

```
API_KEY_KILN=kiln_CLE
```

## pour lancer

`node index.js`
